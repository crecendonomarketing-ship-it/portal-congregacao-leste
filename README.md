# Portal Congregação Leste – São José dos Pinhais (v4)

Nova arquitetura de dados: Programação Semanal + Partes da Reunião (dinâmicas),
Indicadores atualizados com os campos reais (Som e Mídias, Microfone Tribuna,
Indicador/Zoom). Populado com os dados reais de julho–setembro/2026 que você enviou.

## Como publicar

Mesmo processo de sempre: extraia o zip, suba `index.html`, `netlify.toml` e a
pasta `netlify` no seu repositório do GitHub — o Netlify publica sozinho.

## Como atualizar toda semana (rotina simples)

1. **Programação Semanal** (uma linha por semana): preencha Semana, Data, Leitura
   Bíblica, Presidente VM, cânticos, orações — e se houver Reunião Pública nessa
   semana, preencha também Presidente Reunião Pública, Tema, Orador, Congregação
   Visitante e Leitor da Sentinela.
2. **Partes da Reunião**: uma linha por parte. Escolha a Seção (Tesouros / Faça
   Seu Melhor / Nossa Vida Cristã), a Ordem, o Tempo e quem foi designado. Não
   tem limite — quantas partes existirem naquela semana, é o tanto de linhas que
   você cria.
3. **Indicadores e Microfones**: uma linha a cada duas semanas (como no PDF
   original), preenchendo Som e Mídias, Microfone Tribuna, Indicador/Zoom,
   Indicadores 1/2, Microfones 1/2.

Nenhuma dessas etapas cria página nova — são só linhas em bases já existentes.

## Bases descontinuadas

As bases antigas "Reunião Pública" e "Vida e Ministério" foram renomeadas para
"[Descontinuada]" — não são mais usadas pelo site, mas deixei os dados lá por
segurança. Pode apagá-las quando quiser.
